# Notebooks

| # | Notebook | Description |
|---|----------|-------------|
| 1 | Environment Tour | Create environments, inspect observations and actions, render frames |
| 2 | Train and Plot | Launch IMPALA training, plot learning curves |
| 3 | Evaluate and Record | Load checkpoint, run evaluation, record video |
| 4 | Compare Algorithms | IMPALA vs DreamerV2 sample efficiency comparison |
| 5 | Engine Comparison | MuJoCo vs Genesis side-by-side |
| 6 | Model Playground | Load trained model, inspect internals, visualize features |

Run notebooks from the `notebooks/` directory with Jupyter:

```bash
cd notebooks
jupyter lab
```
