# TorchBeast (Vendored)

Pure-Python modules vendored from [torchbeast](https://github.com/facebookresearch/torchbeast) (Facebook Research).

Only the core Python modules are included — no C++ build required.

## Modules

- `core/vtrace.py` — V-trace importance sampling (originally from [scalable_agent](https://github.com/deepmind/scalable_agent))
- `core/environment.py` — Dict-wrapping gym environment with auto-reset
- `core/file_writer.py` — CSV logging and metadata collection
- `core/prof.py` — Timing profiler
